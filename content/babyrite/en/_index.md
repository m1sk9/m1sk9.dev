babyrite is a Discord bot that allows you to easily forward message link content and GitHub PermaLink content to Discord.

- [GitHub - m1sk9/babyrite: A lightweight, fast citation message Discord bot.](https://github.com/m1sk9/babyrite)
- [Status Page](https://status.m1sk9.dev/status/s1)
- [Terms of Service](/babyrite/en/tos)
- [Privacy Policy](/babyrite/en/privacy)

## Features

- Lightweight: babyrite uses distroless as its base image and consists of a very lightweight Docker image.
- Fast: babyrite is developed in Rust and is extremely fast.
- OSS: babyrite is open source and licensed under the MIT License. It also provides a way to self-host and can be published as OSS.
- Easy to use: babyrite is very easy to use and can be deployed in seconds.
