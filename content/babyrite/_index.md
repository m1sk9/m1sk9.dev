babyriteはメッセージリンクのメッセージ / GitHub PermaLink の内容を Discord で簡単に転送できる Discord Bot です．

- [GitHub - m1sk9/babyrite: A lightweight, fast citation message Discord bot.](https://github.com/m1sk9/babyrite)
- [ステータスページ](https://status.m1sk9.dev/status/babyrite)
- [利用規約](/babyrite/tos)
- [プライバシーポリシー](/babyrite/privacy)

## 特徴

- 軽量: babyrite は distroless をベースイメージとして使用しており，非常に軽量な Docker Image で構成されています．
- 高速: babyrite は Rust で開発されており非常に高速．
- OSS: babyrite はオープンソースで MIT License でライセンスされています．また，自分でホストする方法も提供されており，OSS として公開することもできます．
- 使いやすい: babyrite は非常に使いやすく，数秒でデプロイできます．

## 公式ホスト

本来 babyrite は自前でホストすることが想定されていますが，2026年2月からは公式ホストも提供しています．

この公式ホストはコミュニティ版であり，誰でも **完全無料** で利用できます．

公式ホストは [こちら](https://discord.com/oauth2/authorize?client_id=1473215468658757662&permissions=274878024704&integration_type=0&scope=bot) から導入可能です．

なお利用に伴う利用規約・プライバシーポリシーを用意しています．利用前に必ず確認してください．

### 注意点

- コミュニティ版であり，予告なくサービスの停止や再開が行われる可能性があります．
- 完全無料で提供されていますが，利用にあたっては Discord の利用規約を遵守してください．
- 常に最新版を追随します．予告なく機能の追加や削除が行われる可能性があります．
- 公式ホストは babyrite の開発者によって運営されていますが，直接的なサポートは提供されません．問題が発生した場合は GitHub にて報告してください．
- 個別のサーバー設定やカスタマイズは提供されません．
- 最新のステータスについては [status.m1sk9.dev](https://status.m1sk9.dev/status/babyrite) を確認してください．
