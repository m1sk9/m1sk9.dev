## 2. サービスの説明

babyrite は以下の機能を提供する Discord Bot です:

- **メッセージリンク展開**: 同じギルド内で共有された Discord メッセージリンクの埋め込みプレビューを自動生成
- **GitHub パーマリンク展開**: GitHub パーマリンクをシンタックスハイライト付きのコードブロックに展開

## 4. Botの権限

本Botは機能するために以下の権限を必要とします:

- **メッセージを読む/チャンネルを表示**: サーバー内のメッセージリンクを検出するため
- **メッセージを送信**: メッセージプレビューと展開されたコンテンツを投稿するため
- **埋め込みリンク**: メッセージプレビュー用のリッチ埋め込みを表示するため
- **メッセージ履歴を読む**: リンクされたメッセージの内容を取得するため

本Botは，アクセスが許可されたチャンネルのメッセージのみを処理します．

### 5.1 メッセージの処理

- 本Botは**ユーザーデータを収集，保存，または第三者に送信しません**
